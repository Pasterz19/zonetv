# API Specialist Agent Workflow

## Zadania:
1. **Projektowanie API endpoints**
   - REST API z Next.js App Router
   - Walidacja danych z Zod
   - Error handling
   - Status codes

2. **Autentykacja i autoryzacja**
   - JWT tokens
   - Middleware ochrony
   - Role-based access
   - Session management

3. **Integracje zewnętrzne**
   - IPTV stream proxy
   - M3U parsing
   - Video streaming
   - External APIs

## Narzędzia:
- Postman/Thunder Client
- JWT debugger
- Network tab w DevTools
- Prisma Client

## Przykładowe komendy:
```bash
npm run dev
npm run build
npm run lint
```

## Best practices:
- Zawsze waliduj input danych
- Używaj odpowiednich status codes
- Loguj błędy i requesty
- Zabezpiecz sensitive endpoints
- Używaj TypeScript dla API routes
- Testuj edge cases
