# Frontend VOD Agent Workflow

## Zadania:
1. **Tworzenie komponentów React**
   - Komponenty z TypeScript
   - Reużywalne UI komponenty
   - Animacje i interakcje

2. **Stylowanie Tailwind CSS**
   - Responsive design
   - Dark mode
   - Animacje i przejścia
   - Komponenty shadcn/ui

3. **Optymalizacja wydajności**
   - Lazy loading
   - Code splitting
   - Optymalizacja obrazów
   - Next.js optimizations

## Narzędzia:
- React DevTools
- Tailwind IntelliSense
- shadcn/ui
- Framer Motion

## Przykładowe komendy:
```bash
npm run dev
npm run build
npm run lint
```

## Best practices:
- Używaj TypeScript dla wszystkich komponentów
- Trzymaj komponenty małe i reużywalne
- Używaj semantic HTML
- Testuj responsive design
- Optymalizuj dla mobile first
